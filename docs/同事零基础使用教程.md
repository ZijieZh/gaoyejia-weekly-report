# 高爷家周报结论撰写 — 零基础使用教程

> **目标**：让从未用过 Claude Code 的同事，能独立完成"无结论版周报 → 带结论版周报"的全流程。
> **预计时间**：首次配置约 15 分钟，之后每次使用约 2 分钟。

---

## 一、你需要提前准备的

| 项目 | 要求 | 检查方式 |
|------|------|---------|
| 操作系统 | Windows 10/11 | 看桌面左下角开始按钮 |
| Microsoft Excel | 已安装（Office 或 WPS 均可） | 桌面能找到 Excel 图标 |
| Python | 未安装也行，教程里会教 | — |
| 周报文件 | `高爷家云图周报（聚睿）单周.xlsx`（无结论版） | 聚睿提供的原始文件 |

## 二、安装 Claude Code（命令行 AI 工具）

### 步骤 1：打开 PowerShell

按下键盘 `Win + R`，输入 `powershell`，回车。

![打开PowerShell](https://i.imgur.com/placeholder.png)

> 你会看到一个黑底白字的窗口，这就是命令行。后面的所有命令都复制粘贴到这里执行。

### 步骤 2：安装 Node.js（Claude Code 需要它）

在 PowerShell 里逐行执行：

```powershell
# 下载 Node.js 安装器
Invoke-WebRequest -Uri "https://nodejs.org/dist/v20.12.2/node-v20.12.2-x64.msi" -OutFile "$env:TEMP\node-installer.msi"

# 静默安装（不需要你点下一步）
Start-Process msiexec.exe -ArgumentList "/i", "$env:TEMP\node-installer.msi", "/quiet", "/norestart" -Wait

# 验证安装成功
node --version
```

**预期输出**：
```
v20.12.2
```

如果看到版本号（如 `v20.12.2`），说明 Node.js 安装成功。

### 步骤 3：安装 Claude Code

```powershell
npm install -g @anthropic-ai/claude-code
```

等待约 2-3 分钟（取决于网速）。

**验证安装**：
```powershell
claude --version
```

**预期输出**：
```
claude 0.2.xxx
```

### 步骤 4：登录 Claude Code

```powershell
claude auth login
```

会弹出一个浏览器窗口，让你登录 Anthropic 账号（用你平时登录 claude.ai 的 Google 账号即可）。

登录成功后，命令行会显示：
```
Successfully authenticated
```

## 三、安装 Python 环境（数据分析需要）

Claude Code 需要 Python 来读取 Excel 文件。

### 步骤 1：安装 Python

访问 https://www.python.org/downloads/ ，点击黄色按钮 **"Download Python 3.13.x"**。

下载完成后双击安装，**务必勾选"Add Python to PATH"**：

![Python安装勾选PATH](https://i.imgur.com/placeholder.png)

然后点击 **Install Now**。

**验证安装**：重新打开 PowerShell（关掉旧的，再 `Win + R` → `powershell`），执行：
```powershell
python --version
```

**预期输出**：
```
Python 3.13.x
```

### 步骤 2：安装必要的 Python 库

在 PowerShell 里执行：

```powershell
python -m pip install pandas openpyxl pywin32 -i https://pypi.tuna.tsinghua.edu.cn/simple
```

等待约 3-5 分钟。

**验证安装**：
```powershell
python -c "import pandas, openpyxl, win32com; print('OK')"
```

**预期输出**：
```
OK
```

## 四、安装"高爷家周报"Skill

### 步骤 1：克隆 Skill 仓库

在 PowerShell 里执行：

```powershell
cd ~
git clone https://github.com/ZijieZh/gaoyejia-weekly-report.git
```

如果提示 `git` 不是可识别命令，先安装 Git：

```powershell
winget install Git.Git
```

安装完成后**重新打开 PowerShell**，再执行 `git clone`。

### 步骤 2：复制到 Claude Code 的 skills 目录

```powershell
# 创建 skills 目录（如果不存在）
New-Item -ItemType Directory -Force -Path "$env:USERPROFILE\.claude\skills"

# 复制 skill
Copy-Item -Recurse -Force "$env:USERPROFILE\gaoyejia-weekly-report" "$env:USERPROFILE\.claude\skills\gaoyejia-weekly-report"
```

**验证安装**：
```powershell
ls "$env:USERPROFILE\.claude\skills\gaoyejia-weekly-report"
```

**预期输出**：
```
    目录: C:\Users\<你的用户名>\.claude\skills\gaoyejia-weekly-report

Mode                 LastWriteTime         Length Name
----                 -------------         ------ ----
d-----        2026/05/11     14:32                docs
-a----        2026/05/11     14:30           9471 SKILL.md
d-----        2026/05/11     14:30                scripts
```

看到 `SKILL.md` 和 `scripts` 文件夹，说明安装成功。

## 五、实际使用 —— 写周报结论

### 场景：你拿到了本周的无结论版周报

假设文件在桌面，路径是：`C:\Users\<你的用户名>\Desktop\高爷家云图周报（聚睿）单周.xlsx`

### 步骤 1：启动 Claude Code

在 PowerShell 里执行：

```powershell
claude
```

你会看到类似这样的界面：

```
✨ Welcome to Claude Code!

What would you like to work on?

┌───────────────────────────────────────────────────────────────────┐
│  >                                                    │
└───────────────────────────────────────────────────────────────────┘
```

### 步骤 2：告诉 Claude 你要写周报结论

在 `>` 提示符后输入以下任意一种说法：

```
高爷家周报
```

或

```
帮我写高爷家云图周报的结论
```

或

```
处理这个周报文件：C:\Users\<你的用户名>\Desktop\高爷家云图周报（聚睿）单周.xlsx
```

**提示**：把 `<你的用户名>` 换成你电脑的实际用户名。如果不知道，在 PowerShell 里执行 `$env:USERNAME` 就能看到。

### 步骤 3：Claude 自动执行

Claude 会：
1. 读取 Excel 的 5 个 sheet 数据
2. 分析搜索趋势、市场格局、人群资产、渠道数据、SPU 资产
3. 撰写 7 条结论
4. 生成带结论的新文件（文件名末尾会加 `_带结论`）

过程中 Claude 可能会问你问题（比如"数据时间段是 0330-0405 对吗？"），你直接回答即可。

### 步骤 4：拿到结果

Claude 会告诉你输出文件的路径，例如：

```
完成！带结论的文件已保存到：
C:\Users\<你的用户名>\Desktop\高爷家云图周报（聚睿）单周_带结论.xlsx
```

打开这个文件，检查 5 个 sheet 的结论单元格是否都有内容：

| Sheet | 结论位置 | 检查方法 |
|-------|---------|---------|
| 市场搜索趋势 | B3 | 滚动到第3行 |
| 市场格局 | B5（猫砂）、B46（猫粮） | 直接点击这两个单元格 |
| 品牌人群资产 | H2、N3 | 点击 H2 和 N3 |
| 人群和渠道数据 | C2 | 点击 C2 |
| 货品SPU资产新 | J2 | 点击 J2 |

## 六、常见问题

### Q1：Claude Code 启动后没有触发 skill，只是普通对话

**原因**：Skill 文件名或位置不对。

**解决**：
```powershell
# 检查 skill 路径是否正确
ls "$env:USERPROFILE\.claude\skills\gaoyejia-weekly-report\SKILL.md"
```

如果找不到，重新执行步骤四的复制命令。

### Q2：提示"找不到 safe_update_cells"或"win32com 错误"

**原因**：Python 库没装好，或 Excel 进程卡住了。

**解决**：
1. 确认 Python 库已安装（步骤三）
2. 关掉所有 Excel 窗口
3. 在 PowerShell 执行：
   ```powershell
   taskkill /F /IM EXCEL.EXE
   ```
4. 重新在 Claude Code 里请求写结论

### Q3：输出文件打开后格式错乱、颜色丢失

**原因**：Excel 进程卡住，Claude 降级用了 openpyxl 直接保存。

**解决**：
1. 重启电脑（最彻底）
2. 重新运行一次
3. 如果仍有问题，检查 Excel 是否正版激活

### Q4：结论内容感觉不对，数据对不上

**原因**：聚睿提供的周报格式有微调（比如新增了一列）。

**解决**：
- 在 Claude Code 对话里直接告诉它："第 X 行的数据不对，应该是..."
- 或联系维护 skill 的同事更新 SKILL.md

### Q5：Claude Code 提示"权限不足"

**原因**：Windows 权限限制。

**解决**：右键 PowerShell → "以管理员身份运行"，再执行 `claude`。
